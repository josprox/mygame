# Colección de Juegos Python

Una colección de minijuegos clásicos implementados en Python utilizando PySide6 para una interfaz gráfica moderna y atractiva.

## Juegos Incluidos

1.  **El Ahorcado**: Adivina la palabra secreta antes de que se complete el dibujo.
2.  **Cartas Mágicas**: Un truco de magia matemático que adivina el número que estás pensando.
3.  **El Río**: El clásico acertijo de lógica donde debes cruzar al lobo, la gallina y el maíz sin que se coman entre sí.

## Características

-   **Interfaz Gráfica Moderna**: Estilo inspirado en WinUI 3 con modo oscuro.
-   **Arquitectura MVC**: Código organizado y mantenible.
-   **Multi-juego**: Navegación fluida entre el menú principal y los diferentes juegos.

## Requisitos

-   Python 3.8 o superior
-   PySide6

## Ejecución Rápida

```bash
pip install PySide6
python main.py
```

Para más detalles, consulta la [Guía de Instalación](INSTALLATION.md).
